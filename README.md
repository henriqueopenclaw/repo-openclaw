# repo-openclaw — Landing Pages Hub

Monorepo de landing pages da Máquina de Ofertas.

## Estrutura

```
repo-openclaw/
├── lp-protocolo-geladeira/   ← LP 1: Protocolo Geladeira Inteligente
├── lp-projeto-b/             ← LP 2: próxima oferta
└── README.md
```

## Deploy (Vercel)

Cada LP = um projeto Vercel separado com Root Directory apontando pra sua pasta.

- `lp-protocolo-geladeira` → Vercel Project: protocolo-geladeira
- `lp-projeto-b` → Vercel Project: projeto-b
