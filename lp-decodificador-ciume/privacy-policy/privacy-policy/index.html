Política de Privacidade

Última atualização: 28 de fevereiro de 2026

1. Informações que coletamos

Este aplicativo coleta apenas as informações necessárias para sua operação, incluindo dados de uso e interações com a plataforma de anúncios do Meta para fins de gerenciamento de campanhas.

2. Como usamos suas informações

As informações coletadas são utilizadas exclusivamente para:

  • Gerenciamento de campanhas de anúncios no Meta Ads
  • Análise de desempenho de campanhas
  • Melhoria contínua dos serviços

3. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros, exceto quando exigido por lei ou necessário para a operação dos serviços descritos acima.

4. Segurança

Adotamos medidas de segurança adequadas para proteger suas informações contra acesso não autorizado, alteração, divulgação ou destruição.

5. Dados do Meta

Este aplicativo utiliza a API do Meta para gerenciamento de anúncios. O uso dos dados está em conformidade com a Política de Dados do Meta para Desenvolvedores.

6. Contato

Em caso de dúvidas sobre esta política, entre em contato pelo e-mail: contato@ohenriquemelo.com
